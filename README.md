# Cat-and-Mouse

Мини-игра про мышку в лабиринте: сражения с котами, лут из сундуков и прохождение уровней.

## Играть по ссылке

После пуша в GitHub и завершения workflow `Deploy static site to Pages` игра будет доступна по адресу:

`https://<your-username>.github.io/<repo-name>/`

## Локальный запуск

Откройте `index.html` в браузере или поднимите любой статический сервер.
